Key Directories:

  • server/views/: Contains the HTML/EJS templates (Frontend Code).
  • server/public/: Contains static assets (CSS, Images, JS).
  • server/routes/: Handles page routing and data logic.
  • server/data/: JSON files storing the application content.

üé® Frontend Code Overview

The frontend logic is now embedded in the EJS templates. For example, the Gallery page logic:

<% artworks.forEach(artwork => { %>
  <div class="gallery-item">
    <img src="/assets/images/<%= artwork.image %>">
    <h3><%= artwork.title %></h3>
  </div>
<% }); %>

This runs on the server and sends finished HTML to the browser.
